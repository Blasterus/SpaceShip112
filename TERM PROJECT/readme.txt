# SpaceShip!

## Introduction

This project will be called SpaceShip!in which two players attempt to destroy all of their opponents spaceships first.
Made with 112 graphics, it also features 4 distinct types of AI and a 2 player mode. 

## How to run

1. Install the Evil Empire font via 'Evil Empire.otf'
2. Open the folder in VSCode
3. Run tp.py
